# WIM 文件

`WIM`=`Windows Imaging Format`，是[Vista]引入的系统封装方式。有一下特点 :

1. 基于文件的。
2. 硬件无关的。
3. 可以封装多个镜像。比如包含`Windows 7专业版`+`Windows 10专业版`。
4. 文件压缩和硬连接。
